# Health Workspace

This folder is the durable health record for Dill.

- `profile.md`: stable baseline facts
- `goals.md`: active targets and priorities
- `nutrition.md`: eating patterns, macro targets, and useful food history
- `training.md`: programs, exercise preferences, and modification rules
- `supplements.md`: current and past protocols
- `biometrics.md`: measurements, labs, sleep, recovery, and trends
- `logs/YYYY-MM-DD.md`: granular day-level timeline
- `reports/`: summaries of labs, PDFs, plans, or uploaded documents
- `photos/`: manually curated notes tied to progress images when useful
- `whoop/`: auto-synced WHOOP recovery, sleep, cycle, workout, and body-measurement data

Every meaningful health interaction should leave a trail here.
